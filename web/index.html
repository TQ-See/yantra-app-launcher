Under the Hood

Engineering Marvel

Built with precision, designed for performance

Instant Response

Command parsing and execution optimized for sub-millisecond response times. Your launcher should never lag.

Embedded Lua VM

Full LuaJ runtime integrated for scripting. Create automations with HTTP support, system bindings, and custom I/O.

Modular Architecture

Clean separation of concerns with command handlers, theming engine, and scripting runtime as independent modules.

100% Kotlin

Built entirely in modern Kotlin, leveraging coroutines for smooth async operations and null-safety for rock-solid stability.

Fuzzy Search

Levenshtein distance algorithm for intelligent app matching. Type "tube" to launch YouTube. The launcher understands you.

Privacy First

Zero data collection. No analytics. No tracking. Your commands stay on your device. Period.
